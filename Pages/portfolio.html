The Red Maple


During my second-semester training at Keyin College, I completed a web development project focused on building a wildlife educational website.I chose a species, researched it using iNaturalist and other reliable sources, and put together the site with open-licensed materials from the web. All the copywright is original, and I made sure to cite my sources properly in the "about.html" page. I converted the text to HTML, making good use of HTML5 semantic elements. The website includes photos, videos, and audio for a more engaging learning experience. I also applied design using both provided style sheets and class-less CSS frameworks. This project reflects my technical skills and my ability to create a functional and visually appealing website.

Buyify


In the creation of Buyify, my classmates Ellen Dalton, Ethan Miller, and I leveraged our collective JavaScript skills to craft a robust e-commerce platform. Using React, we structured the app with modular components like ProductList, ProductDetails, ShoppingCart, and Checkout. This ensured smooth integration, creating a responsive and interactive interface. We utilized React Router for seamless navigation between pages, enhancing the overall user experience. Additionally, we improved our skills in asynchronous programming and data handling by creating a custom hook for efficient data fetching from the chosen API. This project not only demonstrates our expanded knowledge of JavaScript and React but also reflects our commitment to best practices in web development.
